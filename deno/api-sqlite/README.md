# Deno API

This is a CRUD API made with Deno.

run

```bash
  deno run -A --watch main.ts
```

lint code

```bash
  deno fmt
```

format code

```bash
  deno lint
```

test code

```bash
deno test --allow-env --allow-read  --allow-ffi tests/
deno test  --parallel --allow-env --allow-read  --allow-ffi tests/
```
